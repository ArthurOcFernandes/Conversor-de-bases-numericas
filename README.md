# Bem vindo ao meu conversor de bases numéricas!
Autor: Arthur Fernandes
>  ***O programador é, em sua essência, um solucionador de problemas.***
##
Como um iniciante no mundo da programação, me vejo constantemente em uma situação onde encontro a **essência** de um programador: a solução de problemas. Durante uma aula da faculdade fui apresentado às diferentes bases numéricas utilizadas na computação: _Binário, Hexadecimal e Octal_. Apesar da simplicidade da sua conversão feita em papel vi, ali, uma excelente oportunidade de, além de treinar código, automatizar algo que pode ser exaustivo e repetitivo; por esse motivo resolvi criar este projeto: um conversor de bases numéricas.
##
Esse sistema foi desenvolvido usando Java puro:

* Lógica de programação
* Classes
* Interface controladora
* Padrão de projeto _Strategy_

##
Inicialmente, o sistema roda em terminal e recebe um número decimal inteiro positivo, uma entrada do tipo para conversão e a partir disso calcula e mostra para o usuário o numero convertido.
